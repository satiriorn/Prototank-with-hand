# Robot with manipulator

The project consists of the following elements:
- ESP32 D1 R32;
- DC motor(4);
- Motor driver L298N;
- TP4056;
- 3D printer;
- battery.

Scheme of connecting elements:

The robot is controlled via a Bluetooth module and the Bluetooth Electronics application on Android.
